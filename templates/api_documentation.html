/contrevenants

/contrevenants get

get

Retrieves a list of violations, showing the count of contraventions per establishment, between two specified dates.

get /contrevenants

Retrieves a list of violations, showing the count of contraventions per establishment, between two specified dates.

  • Request
  • Response

Query Parameters

  • du: required(string)

    The start date in ISO 8601 format (YYYY-MM-DD).

  • au: required(string)

    The end date in ISO 8601 format (YYYY-MM-DD).

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

[{"count":1,"etablissement":"BOULANGERIE CHESKIE"},{"count":1,"etablissement":"RESTAURANT A & W"}]

/establishments

/establishments get

get

Retrieves a list of establishments and the count of known infractions for each, sorted in descending order by the number of infractions.

get /establishments

Retrieves a list of establishments and the count of known infractions for each, sorted in descending order by the number of infractions.

  • Response

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

[
  {
    "etablissement": "Restaurant XYZ",
    "infraction_count": 12
  },
  {
    "etablissement": "Cafe ABC",
    "infraction_count": 8
  }
]

/establishments_xml

/establishments_xml get

get

Retrieve a list of establishments with the count of known violations in XML format.

get /establishments_xml

Retrieve a list of establishments with the count of known violations in XML format.

  • Response

HTTP status code 200

Body

Media type: application/xml

Type: any

Example:

<?xml version="1.0" encoding="UTF-8"?>
<establishments>
  <establishment>
    <name>BOULANGERIE CHESKIE</name>
    <violations>5</violations>
  </establishment>
  <establishment>
    <name>RESTAURANT A & W</name>
    <violations>3</violations>
  </establishment>
</establishments>
